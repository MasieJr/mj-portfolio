# Masie Junior Seremu - Personal Portfolio

A retro, paper-themed personal portfolio website built to showcase my software engineering projects, professional experience, and a bit of my personality. 

## 🚀 Tech Stack

*   **Framework:** [Next.js](https://nextjs.org/)
*   **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
*   **Deployment:** [Vercel](https://vercel.com)

## ✨ Features

*   **Custom Paper-Themed UI:** A unique retro aesthetic built from scratch with custom typography and layout rules.
*   **Project Showcase:** Centralized hub highlighting my work in full-stack and mobile application development.
*   **Personal Touches:** Custom graphic elements and sections, including a dedicated nod to trains. 
*   **Responsive Design:** Fully optimized for smooth navigation across mobile and desktop devices.
